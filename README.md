# People Table

### Description

- Implemented the `App` with 2 pages and ability to select a person in the table using external API

### Stack

- HTML (BEM)
- CSS (Bulma)
- JS
- Typescript
- React
- ReactDOM
- React Router
- Fetch
- API ([Mate](https://mate-academy.github.io/react_people-table/api/people.json))

### Tools

- ESlint
- Prettier
- Cypress
- Mochawesome
- Babel
